# MVP Scope

## MVP Hedefi

İlk versiyonda çalışan, skor hesaplayan, rapor üreten assessment platformu.

## MVP Kapsamı

### Dahil

- Organization create
- Assessment create
- 15 kategori
- 30-40 soru
- Tool selector
- Answer save
- Score calculation
- Category score
- Overall score
- Basic recommendations
- Basic roadmap
- HTML report
- Markdown export
- JSON export

### Dahil Değil

- Multi tenant advanced isolation
- Full admin question builder
- PPTX export
- AI recommendation
- Evidence upload
- Advanced benchmark
- SSO

## MVP User Flow

1. Kullanıcı organization oluşturur
2. Assessment başlatır
3. Soruları cevaplar
4. Skor hesaplanır
5. Dashboard görüntülenir
6. Roadmap üretilir
7. Markdown/PDF rapor alınır

## MVP Soru Sayısı

İlk versiyon için 30-40 soru yeterlidir.

## MVP Sonrası

- Admin question bank
- PDF/PPT export
- Evidence upload
- AI supported recommendation
- Benchmark
- Multi customer portal
