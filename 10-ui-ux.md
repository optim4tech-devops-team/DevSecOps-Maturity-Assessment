# UI / UX Design

## Genel Tasarım

Kurumsal, temiz ve danışmanlık kullanımına uygun bir arayüz.

## Ana Ekranlar

### 1. Landing / Dashboard

Kartlar:

- Total Assessments
- Completed Assessments
- Average Maturity
- Top Risk Areas
- Latest Reports

### 2. Assessment List

Kolonlar:

- Assessment Name
- Organization
- Status
- Overall Score
- Created Date
- Completed Date
- Actions

### 3. Assessment Wizard

Sol menü:

- Organization
- SDLC
- Source Control
- CI
- CD
- Release
- DevSecOps
- Testing
- Infrastructure
- Kubernetes
- Observability
- Governance
- Platform Engineering
- AI

Sağ panel:

- Soru
- Cevap seçenekleri
- Tool seçimi
- Note
- Evidence upload
- Save & Next

### 4. Result Dashboard

Görseller:

- Overall score donut
- Category score bars
- Radar chart
- Heatmap
- Top 10 gaps
- Recommendations list
- Roadmap timeline

### 5. Report Preview

Bölümler:

- Executive Summary
- Organization Profile
- Overall Maturity
- Category Analysis
- DevSecOps Gap Analysis
- Tool Inventory
- Roadmap
- Recommendations
- Appendix

## UX Kuralları

- Her kategori tamamlanma yüzdesi göstermeli
- Zorunlu sorular işaretlenmeli
- Conditional questions dinamik açılmalı
- Skorlar canlı güncellenmeli
- Final rapor önizlenebilir olmalı
